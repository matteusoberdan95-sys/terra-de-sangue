# Sprint 17 - Corrida e Impulso

## Objetivo

Corrida com double-tap, impulso frontal com Shift e ataques de entrada.

## Entregas

- [x] Double-tap `A`/`D` ativa corrida (12 stamina, 1.65x velocidade).
- [x] `Shift` impulso frontal (18 stamina, i-frames leves).
- [x] Golpe de entrada durante corrida (`J`).
- [x] Dash attack na janela pos-impulso (`Shift` → `J`).
- [x] Custo de stamina no ataque pesado `K` (8).
- [x] SFX de corrida e impulso.
- [x] Pulo em `Space` (antecipado da Sprint 18).
- [x] Esquiva em `Ctrl` (saiu do Space).
- [x] Facing alinhado ao ataque (`A`/`D`).

## Controles

- `A`/`D` 2x rapido: corrida.
- `Space`: pulo.
- `Shift`: impulso frontal ofensivo.
- `Ctrl`: esquiva com i-frames.
- `J` durante corrida: golpe de entrada.
- `J` apos impulso: dash attack.
